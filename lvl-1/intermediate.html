Specificity and Inheritance

Specificity and inheritance determine how rules are applied. Let's check out the rules below: 

                    h1 { color: blue }                    
                    .main-body { color: red }
and the markup: 
                    <div class="main-body">
                        <h1>What color am I going to be?
                    </div>
